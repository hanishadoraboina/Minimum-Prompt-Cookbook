## 🌟 Overview

**Minimum Prompt Cookbook** is a beautifully designed single-page website created as part of **Module 2 – Prompt Engineering**.

The project demonstrates how well-crafted prompts produce better AI-assisted development by documenting five real prompts, Claude's responses, and the key lessons learned.

Instead of serving as a simple list, the website presents each prompt as an immersive full-screen experience with smooth animations and an elegant developer-inspired interface.

---

## ✨ Features

- 📌 Five carefully selected Claude Code prompts
- 📄 Exact prompt text displayed in styled code blocks
- 🤖 "What Claude Did" explanation for every prompt
- 💡 "What I Learned" insights for every prompt
- 📋 Copy Prompt button with toast notification
- 📱 Fully responsive layout
- 🎨 Glassmorphism-inspired modern UI
- 🚀 Smooth scrolling and animations
- 🎯 Keyboard navigation support
- 📊 Scroll progress indicator
- 🌙 Premium dark theme
